## 代码实验

第四章 `code/` 目录下包含 3 个轻量 Python 示例：

- `calculator_tool.py`：工具函数封装与计算器调用
- `dynamic_planner.py`：动态任务规划与步骤拆解
- `react_structured_parser.py`：ReAct 输出结构化解析

示例运行：

```powershell
python .\第二部分\第四章\code\calculator_tool.py
```
